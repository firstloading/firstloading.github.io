手机termux运行图形界面

Friday, April 24, 2026 1 minute read

termux运行图形界面

可选择环境

  • termux环境
  • termux-proot（其他发行版） 显示方式
  • vnc
  • termux-x11显示

termux环境

一. xfce桌面 + termux-x11显示

I. termux-x11安装

  1. 获取termux-x11显示安装包apk github链接
  2. 安卓设备安装安装包
  3. termux终端运行下面命令
pkg update
pkg install termux-x11 # 安装终端插件

II. 安装xfce 

apt update
apt install x11-repo -y #下载x11源
apt install  xfwm4 xfce4-panel xfce4-settings thunar gvfs xfce4-session xfce4-terminal -y # 安装xfce

III. 运行GUI

termux终端运行以下命令

export DISPLAY=:0 # DISPLAY 变量告诉图形程序连接到哪个 X 服务器（即显示到哪里）
echo $DISPLAY # 检查当前 DISPLAY 值  ， 会输出:0
termux-x11 &  # 启动termux-x11的termux终端插件
startxfce4 & # 启动xfce会话

保持termux终端运行，现在可以去刚安装安卓应用termux-x11使用GUI了

termux-proot环境下安装其他发行版运行xfce并使用termux-x11显示

I. termux-x11安装

  1. 获取termux-x11显示安装包apk github链接
  2. 安卓设备安装安装包
  3. termux终端运行下面命令
pkg update
pkg install termux-x11 # 安装终端插件

II. termux运行其他发行版

安装proot proot-distro

apt update
apt install proot proot-distro -y

安装完成后执行proot-distro list可以列出能proot安装的Linux列表，选择debian发行版， 运行以下命令安装：

proot-distro install debian

进入进入Debian系统

proot-distro login debian

此处省略新建用户miku、添加sudo权限等等正常配置过程。

退出proot-debian,并重新使用新建用户miku重进入Debian系统,精简安装xfce4桌面

proot-distro login debian --user miku
sudo apt update 
sudo apt install xfdesktop4 xfwm4 xfce4-panel xfce4-settings thunar gvfs dbus-x11 xfce4-session xfce4-terminal -y

III. 运行GUI

在termux终端中运行以下命令

termux-x11 :0 &
proot-distro login debian --user miku --shared-tmp # --shared-tmp 链接tmp文件，保证能够将proot-debian的显示内容输出

运行完上述命令后已经进入了proot-debian里面了，接着在proot-debian运行以下命令

export DISPLAY=:0 # DISPLAY 变量告诉图形程序连接到哪个 X 服务器（即显示到哪里）要与前面termux-x11 ：0 &的:0相同
echo $DISPLAY # 检查当前 DISPLAY 值  ， 会输出:0
startxfce4 & # 启动xfce会话

同理， 保持termux终端运行，现在可以去刚安装安卓应用termux-x11里使用GUI了
